Mongoose

1. Create a folder for your project

2. Use "npm install --save express mongoose body-parser" and "npm init" to get node ready for the folder

3. Start a file to bring in Mongoose and create a Schema

4. Create an app file to run Mongoose through

5. Use "node app" to run it
